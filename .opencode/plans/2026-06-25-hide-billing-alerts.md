# Hide Billing Alerts Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `hide_billing_alerts` boolean column to companies table and skip `set_past_due_warning` when it's true.

**Architecture:** A single passive boolean column on `companies` that `ApplicationController#set_past_due_warning` checks before showing the flash alert. No behavioral changes to billing, suspension, or enforcement — those continue regardless of the flag.

**Tech Stack:** Ruby on Rails 8, PostgreSQL

### Task 2: Controller Guard

**Files:**
- Modify: `app/controllers/companies/application_controller.rb`

- [ ] **Step 1: Add `hide_billing_alerts` guard to `set_past_due_warning`**

Insert after the lifecycle check:

```ruby
def set_past_due_warning
  return unless current_company&.lifecycle_status_past_due?
  return if current_company.hide_billing_alerts?
  return unless Time.current.day >= 15

  flash.now[:alert] = "Your account is past due. Please settle outstanding invoices to avoid suspension."
end
```

The `hide_billing_alerts?` predicate is auto-generated by ActiveRecord from the boolean column.

```ruby
# spec/controllers/companies/application_controller_spec.rb
RSpec.describe "set_past_due_warning" do
  let(:company) { create(:company) }
  let(:employee) { create(:employee, company: company) }

  before do
    sign_in(employee.user)
    company.update_columns(lifecycle_status: Company.lifecycle_statuses[:past_due],
                            hide_billing_alerts: true,
                            suspension_at: 1.week.from_now)
    allow(Time).to receive(:current).and_return(Time.new(2026, 6, 20, 10, 0, 0))
  end

  it "skips the past_due warning when hide_billing_alerts is true" do
    get company_dashboard_path(company)
    expect(flash[:alert]).to be_blank
  end
end
```

- [ ] **Step 3: Write controller test — warning shows when hide_billing_alerts is false**

```ruby
it "shows the past_due warning when hide_billing_alerts is false" do
  company.update_columns(hide_billing_alerts: false)
  allow(Time).to receive(:current).and_return(Time.new(2026, 6, 20, 10, 0, 0))
  get company_dashboard_path(company)
  expect(flash[:alert]).to be_present
end
```
